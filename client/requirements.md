## Packages
framer-motion | Smooth animations for quizzes and page transitions
react-markdown | Rendering lesson content which might contain markdown
usehooks-ts | Local storage hook for tracking user progress natively
canvas-confetti | Celebration effect upon completing a lesson
@types/canvas-confetti | Types for canvas-confetti

## Notes
- Progress is tracked purely on the client side using localStorage. No backend auth required.
- Arabic text is rendered using Noto Sans Arabic and Amiri for beautiful typography.
- Uses the existing Shadcn UI components plus custom styled elements for the interactive learning path and quiz.
